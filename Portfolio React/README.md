## 🚀 Como Rodar o Projeto Localmente

Este projeto foi desenvolvido com Node.js v22.2.0.

```bash
# Clone o repositório
git clone https://github.com/lucasgerbasi/portfolio.git

# Acesse a pasta
cd portfolio

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
